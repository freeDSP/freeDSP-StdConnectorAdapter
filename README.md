# freeDSPx Adapter

BOARD STATUS: READY TO BE BUILT

MAIN COORDINATOR: [ssMMnn](https://github.com/ssMMnn)

Adapter for the freeDSP Classic. Adapts the old expansion pinheader to I2S standard connectors. Generates masterclock.
